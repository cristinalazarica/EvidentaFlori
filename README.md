# Tema: Evidența Florilor

Clasa **Floare** conține informații despre produse, precum numele florii, prețul și stocul disponibil, fiind utilizată pentru gestionarea datelor legate de fiecare tip de floare.

Clasa **Client** reține date despre clienți, cum ar fi numele și numărul de comenzi efectuate, oferind o evidență a persoanelor care achiziționează produse.

Clasa **Comanda** realizează legătura între client și floare și conține informații despre numele clientului, numele florii și cantitatea comandată, având rolul de a înregistra fiecare comandă efectuată.

Aplicația permite gestionarea acestor informații prin intermediul unui meniu simplu, oferind posibilitatea de a adăuga, afișa și căuta date într-un mod ușor și organizat.
